# 📱 画面サイズ比較シミュレーター (Screen Size Comparison Simulator)

モバイルデバイスの画面サイズやアスペクト比を、ブラウザ上で**実寸大（1:1スケール）**で比較できるツールです。
最新のスマートフォンからタブレット、さらには「折りたたみ端末（フォルダブル）」の開閉比較にも対応しています。

## ✨ 主な機能
- **実寸校正（キャリブレーション）**: クレジットカードなど身近なものを使って、お使いのモニターに合わせた1mm単位の正確な表示を実現。
- **デバイスデータベース**: 自分のお気に入りの端末や気になる機種を保存し、いつでも呼び出せます。
- **フォルダブル完全対応**: 折りたたみ端末の「開いた状態」と「閉じた状態」をワンボタンで切り替え可能。
- **インテリジェント・スナップ**: デバイスの辺同士が磁石のように吸着し、綺麗に並べて比較できます。
- **PPI自動計算**: 解像度を入力するだけで、画素密度（PPI）を自動で算出。
- **データ管理**: JSON形式でのインポート/エクスポートに対応。

## 🚀 使い方
1. **公開URL**にアクセスします。
2. **「実寸校正モード」**を開き、画面上のカードの大きさを実際のものと合わせます。
3. **「デバイス登録」**から比較したい機種のスペックを入力、または保存リストから選択します。
4. キャンバス上で自由にドラッグ＆ドロップしてサイズ感を比較してください。

## ⚠️ 免責事項
- **表示サイズについて**: 本ツールで表示されるサイズは、入力されたインチ数とアスペクト比に基づく論理値です。ベゼル幅や筐体の外寸、角丸の半径などは考慮されておりません。
- **PPI計算値について**: 画素密度（PPI）はピクセル解像度と対角サイズから標準的な計算式で算出しています。メーカー公表スペックとは、小数点以下の処理や実計測値の違いにより、わずかに異なる場合があります。
- **データの取り扱い**: 本ツールはブラウザのローカルストレージを利用してデータを保存します。ブラウザのキャッシュ消去等によりデータが失われる可能性があるため、重要なデータは定期的に「Export」機能でバックアップを取ることを推奨します。
